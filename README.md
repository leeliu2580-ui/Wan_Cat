# Wan_Cat
猫猫毒舌漫短剧分镜生成（Stray Cat Toxic Sitcom）

## 快速开始
1. 配置环境变量：

```bash
export DASHSCOPE_API_KEY="your-access-key"
```

2. 在 AI Agent 对话框安装 Skill：

```text
安装这个目录下的skill  /path/to/Wan_Cat/skills/stray-cat-toxic-sitcom
```

3. 一键使用（复制填空）：

```text
【点子】（1-3000字）：
【金句】（可选，逗号分隔）：
【模式】更喜剧 / 平衡 / 更催泪（默认：平衡）：
【张数】3-12（默认：8）：
【画幅】9:16 / 16:9 / 3:4（默认：9:16）：
【清晰度】1K / 2K（默认：1K）：
【猫设定】狸花 / 橘猫 / 黑猫 / 奶牛猫（默认：狸花）：
【参考图】无 / 本地路径 / URL（最多9张，默认：无）：
```
